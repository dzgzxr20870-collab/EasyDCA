นโยบายความเป็นส่วนตัว (Privacy Policy)

ปรับปรุงล่าสุด: 17 กรกฎาคม 2569 (2026-07-17)

EasyDCA ("เรา") เป็นบริการช่วยบันทึกและติดตามการลงทุนแบบถัวเฉลี่ยต้นทุน (Dollar-Cost Averaging) ผ่าน LINE Official Account และหน้าเว็บ Dashboard ประกอบ นโยบายนี้อธิบายว่าเราเก็บ ใช้ และดูแลข้อมูลส่วนบุคคลของคุณอย่างไร ตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA)

1. ข้อมูลที่เราเก็บ

  • ข้อมูลโปรไฟล์ LINE — LINE User ID, ชื่อที่แสดง (Display Name), รูปโปรไฟล์ — ได้รับจาก LINE ตอนคุณเพิ่มเพื่อนหรือ Login ผ่าน LIFF เพื่อผูกบัญชีของคุณกับข้อมูลการลงทุนที่คุณบันทึก
  • ข้อมูลธุรกรรมการลงทุนที่คุณพิมพ์เข้ามาเอง — เช่น ชื่อสินทรัพย์ จำนวนหน่วย ราคาต่อหน่วย วันที่ทำรายการ (ซื้อ/ขาย) — ใช้เพื่อคำนวณพอร์ตการลงทุน กำไร/ขาดทุน และประวัติของคุณเท่านั้น
  • การตั้งค่าการแจ้งเตือนและเป้าหมายการลงทุน — เช่น วันที่/ความถี่ในการแจ้งเตือน DCA ที่คุณตั้งไว้เอง
  • รูปภาพสลิปการโอนเงิน — สำหรับผู้ใช้ที่สมัคร Premium ผ่าน PromptPay QR เก็บไว้เพื่อให้ผู้ดูแลระบบตรวจสอบยอดก่อน อนุมัติเท่านั้น
  • ข้อมูลการชำระเงิน Premium — คำขอชำระเงิน (จำนวนเงิน, วันเวลา, สถานะ) — เราไม่เก็บข้อมูลบัตรเครดิต/เดบิต หรือข้อมูลบัญชี ธนาคารของคุณ การยืนยันการชำระเงินทำโดยผู้ดูแลระบบตรวจสอบยอดเงินเข้า บัญชีด้วยตนเอง

2. เหตุผลทางกฎหมายและวัตถุประสงค์การใช้ข้อมูล

เราเก็บและใช้ข้อมูลของคุณโดยอาศัยฐานทางกฎหมาย "ความจำเป็นในการปฏิบัติ ตามสัญญา" (จำเป็นต้องใช้ข้อมูลเพื่อให้บริการที่คุณสมัครใช้งาน) และ "ความยินยอมของคุณ" (Consent) สำหรับข้อมูลบางส่วน โดยมีวัตถุประสงค์ดังนี้

  • แสดงผลพอร์ตการลงทุน กำไร/ขาดทุน และประวัติธุรกรรมของคุณ
  • ส่งการแจ้งเตือนตามที่คุณตั้งค่าไว้ (เช่น แจ้งเตือน DCA)
  • ยืนยันตัวตนและจำกัดสิทธิ์การใช้งานตามแผน (Free/Premium)
  • ตรวจสอบและอนุมัติการชำระเงิน Premium
  • ปรับปรุงคุณภาพบริการ
หมายเหตุ: ข้อมูลการใช้งานของคุณอาจถูกนำไปวิเคราะห์เพื่อ ปรับปรุงระบบในรูปแบบที่ไม่ระบุตัวตน (Anonymized) เช่น สถิติภาพรวมของการ ใช้ฟีเจอร์ต่างๆ — ข้อมูลรูปแบบนี้ไม่สามารถย้อนกลับไปหาตัวคุณได้อีก

3. การเปิดเผยข้อมูลต่อบุคคลที่สาม

เราไม่ขายหรือให้เช่าข้อมูลส่วนบุคคลของคุณแก่บุคคลที่สาม ข้อมูลของคุณ ถูกจัดเก็บบนผู้ให้บริการโครงสร้างพื้นฐานที่เราใช้งานเท่าที่จำเป็นต่อการ ให้บริการเท่านั้น ได้แก่ Supabase (ฐานข้อมูลและที่เก็บไฟล์) และ Railway (เซิร์ฟเวอร์) ซึ่งมีนโยบายความปลอดภัยของตนเอง เราใช้ LINE Messaging API และ LINE Login (LIFF) เพื่อยืนยันตัวตนและส่งข้อความตามมาตรฐานที่ LINE กำหนด และใช้ Claude Vision API (Anthropic) เพื่ออ่านข้อมูลจากรูปสลิปที่ คุณส่งมาโดยอัตโนมัติสำหรับผู้ใช้ Premium เท่านั้น

4. สิทธิ์ของคุณตาม PDPA

คุณมีสิทธิ์ดังต่อไปนี้เกี่ยวกับข้อมูลส่วนบุคคลของคุณ:

  • สิทธิ์ขอเข้าถึงข้อมูล (Access) — ขอดูข้อมูลที่เรา เก็บเกี่ยวกับคุณได้ทุกเมื่อผ่านหน้าเว็บ Dashboard หรือติดต่อเราตาม ช่องทางด้านล่าง
  • สิทธิ์ขอแก้ไขข้อมูล (Rectification) — หากข้อมูล ไม่ถูกต้อง (เช่น ชื่อที่แสดงผิด) แจ้งเราเพื่อแก้ไขได้
  • สิทธิ์ถอนความยินยอม (Withdrawal) — คุณสามารถหยุด ใช้งานและถอนความยินยอมได้ทุกเมื่อ โดยผลจะเทียบเท่ากับการขอลบข้อมูล ด้านล่าง
  • สิทธิ์ขอรับข้อมูล (Portability) — ขอสำเนาข้อมูล ธุรกรรมของคุณในรูปแบบที่อ่านได้ (เช่น ผ่านฟีเจอร์ "ส่งออกรายงาน" ใน LINE Chat หรือติดต่อขอโดยตรง)
  • สิทธิ์ขอให้ลบข้อมูล (Erasure) — ดูวิธีใช้สิทธิ์นี้ ในหัวข้อถัดไป

วิธีขอลบข้อมูล (Erasure) ด้วยตัวเอง

พิมพ์คำว่า ลบข้อมูล ในหน้าแชท LINE ของ EasyDCA ระบบจะอธิบาย ผลกระทบและให้คุณกดยืนยันอีกครั้งก่อนดำเนินการจริง (ป้องกันการกดผิดพลาด) เมื่อยืนยันแล้ว ระบบจะดำเนินการทันที ดังนี้:

  • ข้อมูลที่ระบุตัวตนได้ (LINE User ID, ชื่อที่แสดง, รูปโปรไฟล์) จะถูก ลบ/แทนที่ด้วยข้อมูลที่ไม่ระบุตัวตนอย่างถาวร ไม่สามารถย้อนกลับได้
  • รูปสลิปการโอนเงินทั้งหมดของคุณจะถูกลบออกจากระบบจริง
  • คุณจะไม่สามารถ Login เข้าใช้บัญชีเดิมได้อีก (หากเข้าใช้งานผ่าน LINE เดิมอีกครั้ง ระบบจะสร้างบัญชีใหม่ที่ไม่มีประวัติเดิมให้แทน)
  • ประวัติธุรกรรมและการชำระเงินจะยังถูกเก็บไว้ แต่ในรูปแบบที่ ไม่ระบุตัวตน (Anonymized) — เราเก็บส่วนนี้ไว้เพราะกฎหมายไทย กำหนดให้ต้องเก็บหลักฐานทางบัญชี/การเงินไว้ระยะหนึ่ง (โดยทั่วไป ประมาณ 5 ปี) และเพื่อความถูกต้องของบัญชีของผู้ใช้รายอื่นที่อาจเกี่ยว ข้องกัน ข้อมูลส่วนนี้จะไม่สามารถเชื่อมโยงกลับมาที่ตัวคุณได้อีกต่อไป
  • หากคุณมีคำขอชำระเงินที่ยังไม่ได้รับการตรวจสอบค้างอยู่ ระบบจะแจ้ง เตือนก่อนว่าการลบข้อมูลจะทำให้ผู้ดูแลระบบตรวจสอบไม่ได้อีกว่ารายการนั้น เป็นของใคร ก่อนให้คุณตัดสินใจยืนยัน

5. ความปลอดภัย

เราใช้มาตรการที่เหมาะสมในการป้องกันข้อมูลของคุณ เช่น การเข้ารหัสการ เชื่อมต่อ (HTTPS) และการยืนยันตัวตนด้วย JSON Web Token (JWT) ก่อนเข้าถึง ข้อมูลส่วนตัวใดๆ ผ่านหน้าเว็บ Dashboard

6. ระยะเวลาการเก็บข้อมูล

เราเก็บข้อมูลของคุณไว้ตลอดอายุการใช้งานบัญชี จนกว่าคุณจะใช้สิทธิ์ขอลบ ข้อมูลตามหัวข้อ 4 อย่างชัดแจ้ง ยกเว้นข้อมูลที่กฎหมายอื่นบังคับให้ต้องเก็บ รักษาต่อในรูปแบบไม่ระบุตัวตนตามที่อธิบายไว้ข้างต้น

7. การเปลี่ยนแปลงนโยบาย

เราอาจปรับปรุงนโยบายนี้เป็นครั้งคราว การเปลี่ยนแปลงจะแสดงวันที่ปรับปรุง ล่าสุดไว้ด้านบนของหน้านี้เสมอ

8. ติดต่อเรา

หากมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ หรือต้องการใช้สิทธิ์ใดๆ ตามหัวข้อ 4 นอกเหนือจากการลบข้อมูลด้วยตัวเอง สามารถติดต่อได้ผ่าน LINE Official Account ของ EasyDCA
