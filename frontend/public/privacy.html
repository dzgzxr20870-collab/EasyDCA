นโยบายความเป็นส่วนตัว (Privacy Policy)

ปรับปรุงล่าสุด: 4 กรกฎาคม 2569 (2026-07-04)

EasyDCA ("เรา") เป็นบริการช่วยบันทึกและติดตามการลงทุนแบบถัวเฉลี่ยต้นทุน (Dollar-Cost Averaging) ผ่าน LINE Official Account และหน้าเว็บ Dashboard ประกอบ นโยบายนี้อธิบายว่าเราเก็บ ใช้ และดูแลข้อมูลของคุณอย่างไร

1. ข้อมูลที่เราเก็บ

  • ข้อมูลโปรไฟล์ LINE — LINE User ID, ชื่อที่แสดง (Display Name), รูปโปรไฟล์ — ได้รับจาก LINE ตอนคุณเพิ่มเพื่อนหรือ Login ผ่าน LIFF เพื่อผูกบัญชีของคุณกับข้อมูลการลงทุนที่คุณบันทึก
  • ข้อมูลธุรกรรมการลงทุนที่คุณพิมพ์เข้ามาเอง — เช่น ชื่อสินทรัพย์ จำนวนหน่วย ราคาต่อหน่วย วันที่ทำรายการ (ซื้อ/ขาย) — ใช้เพื่อคำนวณพอร์ตการลงทุน กำไร/ขาดทุน และประวัติของคุณเท่านั้น
  • ข้อมูลการชำระเงิน Premium — คำขอชำระเงิน (จำนวนเงิน, วันเวลา, สถานะ) สำหรับผู้ใช้ที่สมัครสมาชิก Premium ผ่าน PromptPay QR — เราไม่เก็บข้อมูลบัตรเครดิต/เดบิต หรือข้อมูลบัญชีธนาคารของคุณ การยืนยันการชำระเงินทำโดยผู้ดูแลระบบตรวจสอบยอดเงินเข้าบัญชีด้วยตนเอง

2. เราใช้ข้อมูลของคุณเพื่อ

  • แสดงผลพอร์ตการลงทุน กำไร/ขาดทุน และประวัติธุรกรรมของคุณ
  • ส่งการแจ้งเตือนตามที่คุณตั้งค่าไว้ (เช่น แจ้งเตือน DCA)
  • ยืนยันตัวตนและจำกัดสิทธิ์การใช้งานตามแผน (Free/Premium)
  • ปรับปรุงคุณภาพบริการ

3. การเปิดเผยข้อมูลต่อบุคคลที่สาม

เราไม่ขายหรือให้เช่าข้อมูลส่วนบุคคลของคุณแก่บุคคลที่สาม ข้อมูลของคุณ ถูกจัดเก็บบนผู้ให้บริการโครงสร้างพื้นฐานที่เราใช้งาน ได้แก่ Supabase (ฐานข้อมูล) และ Railway (เซิร์ฟเวอร์) ซึ่งมีนโยบายความปลอดภัยของตนเอง เราใช้ LINE Messaging API และ LINE Login เพื่อยืนยันตัวตนและส่งข้อความ ตามมาตรฐานที่ LINE กำหนด

4. สิทธิ์ของคุณ

คุณสามารถขอดู แก้ไข หรือขอให้ลบข้อมูลส่วนบุคคลของคุณได้ โดยติดต่อเราตาม ช่องทางด้านล่าง การลบบัญชีจะทำให้ไม่สามารถใช้งานบริการต่อได้ และข้อมูล ธุรกรรมที่เคยบันทึกไว้จะถูกลบตามคำขอ (ยกเว้นข้อมูลที่กฎหมายกำหนดให้ต้อง เก็บรักษา เช่น หลักฐานการชำระเงิน)

5. ความปลอดภัย

เราใช้มาตรการที่เหมาะสมในการป้องกันข้อมูลของคุณ เช่น การเข้ารหัสการ เชื่อมต่อ (HTTPS) และการยืนยันตัวตนด้วย JSON Web Token (JWT) ก่อนเข้าถึง ข้อมูลส่วนตัวใดๆ ผ่านหน้าเว็บ Dashboard

6. การเปลี่ยนแปลงนโยบาย

เราอาจปรับปรุงนโยบายนี้เป็นครั้งคราว การเปลี่ยนแปลงจะแสดงวันที่ปรับปรุง ล่าสุดไว้ด้านบนของหน้านี้เสมอ

7. ติดต่อเรา

หากมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ สามารถติดต่อได้ผ่าน LINE Official Account ของ EasyDCA
